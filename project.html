Multilingual Assistive Model for Visually Impaired Users

This project aims to support visually impaired users by integrating image captioning, translation, and text-to-speech. The model generates descriptive captions, translates them into multiple Indic languages, and converts the translations into speech using cutting-edge tools.

Frameworks, Tools, Libraries Used: Pytorch, Blip's model, Transformers, IndicTrans2, MarianMT, gTTS API, and Facebook’s MMS-TTS

Plant Disease Detection using Big Data Tools

This project involved training a 9-layer Convolutional Neural Network (CNN) model on a 55 GB dataset of images from over 20 plant species. Industry-standard Big Data frameworks such as Hadoop HDFS and Apache Spark were used to manage and process the data. The workflow, including data pre-processing and model training, is documented in an IPython notebook on my GitHub .The neural network was deployed on Big Data clusters at the Indian Institute of Technology Bhilai.

Frameworks, Tools, Libraries Used: Pyspark, TensorFlow, Apache Spark, SparkML, Keras, and OpenCV
